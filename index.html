1) Description of how the dashboard answers the questions

The first question asks us to find out the top 10 games of all time. In my Dashboard, I have created a bar chart that indicates the games sorted by their Global Sales. I have used Global_Sales to determine the best 10 games. The next question expects us to break down the genre sales per region. For this purpose,I have created an interactive bar-chart. You can toggle the buttons present on the top left of the dashboard to see the proportion of sales across genres in North America,Europe and Japan. I had scraped the required data from the original video_games.csv file, using pandas. The python code is present in util.py and the data is present in the file genre_region.csv. ( Note: Due to lack of time and knowledge of javascript, I had to hardcode the values from genre_region.csv file into the main.js file(to get a dictionary of values). You can check the values by running util.py and verify that they are the same). The third question asks us to provide the best publisher for each genre. I created a scatterplot which indicates the best publisher in each genre based on the Global_Sales for their best game. I have introduced a tooltip, which would indicate the global sales when you hover over the points. I had scraped the data from video_sales.csv using util3.py into genre_publisher.csv.

2) Three Reasons why D3 was helpful

i) The Flexibility that it offers in terms of making unique visualizations using my own data. The eclectic variety of graphs on the d3-gallery is quite amusing. ii) The Ease with which we can add interactivity to the graphs is quite appreciable. Javascript is relatively an easy scripting language to work with. iii) We can add smooth transitions for updating elements. This is something I appreciated a lot while coding my pie-chart. As I was toggling over the buttons, D3 was smoothly generating the pie-chart without much lag.

3) Three Reasons why D3 may not be the best option

i) I personally had issues with the compatibility between version 5 and version 4. Functions like filter were throwing an error when used with version 5. ii) From some research that I had done online, some functions in D3 apparently are not supported in older browsers. iii) D3 does not automatically generate any graphs like Tablueau. This can be a disadvantage when you do not want a lot of customization.
